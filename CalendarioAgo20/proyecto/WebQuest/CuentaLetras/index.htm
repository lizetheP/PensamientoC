La disponibilidad de computadoras con capacidades para manipular cadenas ha originado algunosmétodos para analizar los escritso de grandes autores. Se ha puesto mucha atención en el hecho de si William Shakespeare en realidad vivió. Algunos estudiosos creenque existe suficiente evidencia para demostrar que en realidad Christopher Marlowe escribió los escritos adjudcados a Shakespeare Los investigadores aplican dos métodos principalmente para analizar los textos mediante una computadora:

1. Consiste en contar el número de veces que se repite una letra en el texto, por ejemplo en el siguiente texto: "Ser o no ser, esa es la cuestión" se utilizan cinco letras S, cinco letras E, dos letras R, tres letras O, dos letras N, dos letras A, una letra L, una letra C, una letra U, una letra T y una letra I.
2. Consiste en contar el número de palabras que contengan una sola letra, dos letras y así sucesivamente. Por ejemplo en la línea de texto anterior: "Ser o no ser, esa es la cuestión" se utiliza una palabra de una letra, tres palabras de dos letras, tres palabras de tres letras y una de ocho letras.
