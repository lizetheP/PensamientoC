Funciones predefinidas en Python  
. 

 Objetivo:  
    • Conocer las funciones predefinidas en Python.

  Modalidad:  
    • Sesión plenaria.
 
Instrucciones:

    • Entra a Thonny.

    • Importa la biblioteca math con la siguiente instrucción: import math

    • Después solicita ayuda de la biblioteca de funciones math: help(math)

                     

    • Localiza en el texto las funciones que requieres y pruebalas.                         

                   Por ejemplo:  

    • Para probar la función sqrt haz algo como esto: math.sqrt(4.5)  

                     

                        

,
  
 Ejercicios:

 

 

                      Escribe la sentencia en Python que implementa las siguientes expresiones matemáticas:

    • La raíz cuadrada de B2+4AC.

    • La raíz cuadrada de X + 4Y3

    • La raíz cúbica del producto de X y Y.  

 
  Recursos:


    • En este enlace https://docs.python.org/3/library/math.html también se pueden consultar las funciones predefinidas.
