TC1028. Pensamiento computacional para ingeniería 
.  

Tarea individual: Funciones   
.
.
 
 Objetivos:  
    • Aplicar los conocimientos sobre funciones.
    • Reconocer la importancia del uso de funciones para el diseño de un programa.

 Modalidad:  
    • Individual.
 
 Instrucciones:
  
    • Implementa las siguientes funciones, recuerda que las funciones se definen antes de la función del main y del script principal.
    • La función main () debe mandar llamar a la función que imprime el menú correspondiente y de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas. Utiliza el estatuto de control if anidado. Recuerda que la captura de datos debe ser realizada en la sección del main. Valida opciones incorrectas.
    • En el script principal manda llamar a la función main.

                   

 
  Ejercicios:

  Construye las funciones para calcular las longitudes o superficies de cada una de las siguientes figuras:

    • La función arco (radio, angulo) que recibe el radio y el ángulo. Utiliza la constante math.pi de la librería math.

      Casos de prueba:

      Input: Número de opción (1. Arco)
      Input: Valor del radio (real)
      Input: Valor del angulo (real)
      Output: Superficie de la figura (real con dos decimales)

      Input: 1
      Input: 5.5
      Input: 45
             Output: 4.32

      Input: 1
      Input: 8.5
      Input: 270
      Output: 40.06

    • La función poligono (l, a) que recibe el lado (l) y el radio (a).

       

      Casos de prueba:

      Input: Número de opción (2. Poligono)
      Input: Valor del lado l (real)
      Input: Valor del radio a (real)
      Output: Superficie de la figura (real con dos decimales)

      Input: 2
      Input: 7.2
      Input: 5.3
      Output: 114.48

      Input: 2
      Input: 8.5
      Input: 3.4
      Output: 86.70


    • La función triangulo (b, h) que recibe la base (b) y la altura (h).

       

      Casos de prueba:

      Input: Número de opción (3. Triángulo)
      Input: Valor de la base b (real)
      Input: Valor de la altura h (real)
      Output: Superficie de la figura (real con dos decimales)

      Input: 3
      Input: 7.4
      Input: 5.9
      Output: 21.83

      Input: 3
      Input: 6.2
      Input: 4.7
      Output: 14.57


    • La función figuras () que despliegue el siguiente menú en pantalla:

        1. Arco
        2. Polígono
        3. Triángulo
         4. Salir

      Casos de prueba:

      Input: Número de opción (4. Salir)
      Output: Mensaje "Adios"

      Input: 4
      Output: Adios

Casos de prueba:

Input: Número de opción (Opción inválida)
Output: Mensaje "Opcion_invalida"

Input: -3
Output: Opcion_invalida

Input: 8
Output: Opcion_invalida

    • La función main () que utilice la función figuras y de acuerdo a la opción seleccionada por el usuario mande llamar a la función correspondiente para calcular la superficie elegida. Utiliza el estatuto de control if anidado. Recuerda que la captura de datos debe ser realizada en la sección del main. Valida opciones incorrectas.
    • En el script principal manda llamar a la función main.
