Estructura condicional if

Objetivo

Conocer la estructura de control if (if, elif, else anidados) y los operadores relacionales.

Revisión de autoestudio

Desarrolla el código para cada uno de los siguientes problemas:

  • Dado un valor n, convertirlo a su valor absoluto.
  • Dado un valor n, determinar si es par o impar.
