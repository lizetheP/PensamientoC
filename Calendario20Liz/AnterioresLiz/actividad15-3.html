Actividad: Desarrollando programas en Python No. 10: Archivos.

info_outline Actividad: Desarrollando programas en Python No. 10: Archivos.

Desarrollarás la solución de algunos problemas relacionados con archivos.

group Modalidad

Individual

check Objetivos de aprendizaje
  • Aplicar los conceptos de flujo de datos en la solución de problemas.
list Instrucciones
  • Analiza cada uno de los ejercicios que se presentan a continuación. Diseña el algoritmo para la solución de cada uno de los ejercicios. Recuerda incluirlo con comentarios en el código.
  • Escribe un programa en Python para cada uno de los ejercicios siguiendo tu algoritmo y haz las modificaciones necesarias para resolver el problema.
  • Entrega, en la sección de "Envío de Tareas", los archivos .c que contengan el código fuente del programa.
  • La actividad será evaluada usando la siguiente rúbrica.
  • Problemas:

    Usando como base el programa desarrollado en la sesión anterior, agrega las siguientes funciones:

  • Ejercicio 1.

    Desarrolla el procedimiento reporte_autor que recibe el nombre de un autor. El procedimiento crea un archivo de texto llamado "autor.txt" en donde se guardan los nombres de todos los libros que ha escrito un autor determinado (y que no estén marcados como borrados). El encabezado del procedimiento podría ser el siguiente:

    void reporte_autor (char nombre_autor[])

  • Ejercicio 2.

    Desarrolla el procedimiento borra_libro que recibe el identificador del libro a borrar. El procedimiento debe marcar como borrado el libro, sí y solo sí el libro no está prestado. El encabezado del procedimiento podría ser el siguiente:

    int borra_libro ( int id )

  • Ejercicio 3.

    Desarrolla el procedimiento reporte_borrados. El procedimiento crea un archivo de texto llamado "borrados.txt" en donde se guardan los nombres de todos los libros que han sido borrados. El encabezado del procedimiento podría ser el siguiente:

    int reporte_borrados ()

    En el main() agrega las instrucciones necesarias para utilizar las nuevas funciones.

    Incluye el algoritmo como comentario en el programa.
