Listas anidadas o matrices

Objetivo

Entender el concepto de listas anidadas o matrices.

Revisión de autoestudio

1. ¿Qué imprimiría el siguiente código?

El primero usa los iteradores inatos del ciclo for, donde guarda un arreglo de 2 elementos en cada vuelta [0,1] y columna guarda un elemento del arreglo en cada vuelta 0.

matriz = [[0,1],[2,3],[4,5]]
valor = 0
for linea in matriz:
    for elemento in linea:
        valor = valor + elemento #usamos los iteradores
print(valor)

2. ¿Qué imprimiría el siguiente código?

El segundo usa índices directamente sobre la matriz o arreglo multidimensional. Donde i es usado como índice del arreglo exterior de 3 elementos [0,1],[2,3],[4,5] y j es usada como el índice del interior de cada arreglo de 2 [0,1]

matriz = [[0,1],[2,3],[4,5]]
valor = 0
#indice i (renglones)
for i in range ( 0, len( matriz ) ):
    #indice j (columnas)
    for j in range ( 0, len( matriz[i] ) ):
        valor = valor + matriz[i][j]  #usamos los indices
print(valor)
