    • La función aproximacionPI(n) que recibe un valor entero mayor o igual a uno y regresa una aproximación de PI, utilizando la siguiente fórmula:

(4/1) - (4/3) + (4/5) - (4/7) + (4/9) - (4/11) .........

El numero que recibes te dice cuantos elementos calcula, entre más elementos mejor aproximación.

Ejemplos:

aproximacionPI(3) = (4/1) - (4/3) + (4/5)

aproximacionPI(6) = 4/1) - (4/3) + (4/5) - (4/7) + (4/9) - (4/11)     

    • La función factorial(n) que recibe un número n y calcula el factorial de n.

El factorial de un número n = 1 * 2 * 3 * ... * n. 

Ejemplos:

factorial(2) = 1 * 2 = 2

factorial(3) = 1 * 2 * 3 = 6

factorial(4) = 1 * 2 * 3 * 4 = 24
