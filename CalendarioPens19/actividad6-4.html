info_outline Autoestudio sobre funciones y procedimientos: Estatutos de repetición.

Realizarás el autoestudio que presenta los conceptos de procedimientos.

group Modalidad

Individual

check Objetivos de aprendizaje
  • Profundizar los conceptos de función y procedimiento, variable global y local.
  • Diferenciar entre una función y un procedimiento, una variable global y una local.
list Instrucciones

  • Estudia de uno de los siguientes libros la secciones indicadas:

    Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2010.

    1. Resultados de una función. (pp. 194)
    2. Llamada a una función. (pp. 195)
    3. Parametros de una función. (pp. 201)
    4. Bibliotecas. (pp. 212)

    Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2014.

    1. Resultados de una función. (pp. 180)
    2. Llamada a una función. (pp. 181)
    3. Parametros de una función. (pp. 187)
    4. Concepto y uso de funciones de biblioteca. (pp. 198)

  • Responde las siguientes preguntas:

    1. 1. ¿Cuál es la diferencia entre una función y procedimiento?
    2. 2. ¿Cuando conviene usar una función y cuando un procedimiento?
    3. 3. ¿Como se saca la información de un procedimiento?
    4. 4. ¿En un programa como podrías cuando se manda a llamar una función de un procedimiento?
