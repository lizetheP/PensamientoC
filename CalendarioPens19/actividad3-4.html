  • Aplicar la estructura de control if (if, if-else, if anidados) y los operadores relacionales.
list Instrucciones
  • Analiza cada uno de los ejercicios que se presentan a continuación. Identifica cual sería el algoritmo para la solución de cada uno de los ejercicios.
  • Escribe un programa en Python para cada uno de los ejercicios. Recuerda basarte en el algoritmo que generaste.
  • Entrega, en la sección de "Envío de Tareas", los archivos que contengan los programas en C.
  • La actividad será evaluada usando la siguiente rubrica.
  • Problemas:

    Utiliza el siguiente esquema como apoyo para desarrollar tus soluciones:


  • Ejercicio 1

    Desarrolla un programa en Python que determine la velocidad (v) alcanzada por un automóvil en una distancia (d) y un tiempo (t). Estas últimas 2 variables son dadas por el usuario.


    casos de prueba F(d, t).
    a) F(10, 2) = 5
    b) F(0, 2) = 0
    c) F(10, 0) = operación inválida

    Incluye el algoritmo como comentario al inicio del programa.

  • Ejercicio 2

    Desarrolla un programa en Python que despliegue el resultado de evaluar la siguiente función con el valor dado por el usuario.


    casos de prueba F(d, t).
    a) F(12) = 37
    b) F(0) = 1
    c) F(-3) = 18 -12 + 2 = 8
    d) F(-1) = 2

    Incluye el algoritmo como comentario al inicio del programa.

  • Ejercicio 3

    Desarrolla un programa en Python que imprima el mensaje apropiado para una temperatura dada por el usuario. Los mensajes a imprimir serán los siguientes:

    1. 100 grados o más => despliega "Vapor".
    2. 30 grados o más, pero menos de 100 grados => despliega "Caliente".
    3. 0 grados o más, pero menos de 30 grados => despliega "Fria".
    4. -273 grados o más, pero menos de 0 grados => despliega "Congelada".
    5. Menos de -273 => despliega "Temperatura inexistente".

    casos de prueba.
    a) F(120) = "Vapor"
    b) F(0) = "Fria"
    c) F(-3) = "Congelada"
    d) F(-274) = "Temperatura inexistente"

    Incluye el algoritmo como comentario al inicio del programa.

  • Programa Abierto:

    Desarrolla un programa en Python con el tema de tu preferencia (diferente a los que se plantean en el laboratorio) que involucre al menos 3 niveles de decisión, usa todas las formas de If else.

    Este programa es tuyo, úsalo para hacer cosas interesante y retadoras no cosas fáciles.
