info_outline Actividad: Instalación del ambiente de programación.

Realizarás la instalación del ambiente de programación que estaremos usando a lo largo del semestre.

check Objetivos de aprendizaje
  • Instalación del ambiente de desarrollo.
list Instrucciones
  • Baja e instala Thonny que es un IDE con compilador para principiantes.
  • (opcional) Si ya tienes experiencia y quieres usar herramientas más complejas entonces:
    Baja e instala Python 3 que es el compilador independiente.
    Baja e instala un IDE (Integrated Development Environment) que es editor de texto para programar. IDEs populares por ser muy flexibles son notepad++ para windows o sublime para mac o linux.
  • Si no tienes un computadora propia o prefieres programar en tu navegador web, entonces crea una cuenta en Repl.it que es una página de internet que ya tiene instalado el compilador y el editor.
