Uso de operadores

 
Ejemplo Descripción
Aritméticos:
x = x * ( 3 / 5) - y;
z = (30 % 4 ) / 3;
Lógicos y relacionales:
( A > B ) || (A < C)  Es verdadero en caso de que A sea mayor que B ó menor que C
!(A >= B) && (X < 0) Es verdadero en caso de que A no sea mayor o igual a B (es decir, es menor) y X es menor que cero.
X == 5  Se pregunta si X es igual a 5 (no es lo mismo que asignar).
Asignación:
X = X + 1;  Incrementa en uno a X.
X = 5; Asigno el valor de 5 a X.
++X;  incremento el valor de X en uno.
