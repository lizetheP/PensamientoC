Avance 3. Modelando la solución de mi proyecto final 
.

Objetivo:

Crear los algoritmos de tu proyecto final.


Forma de trabajo:

Actividad colaborativa


   Instrucciones:
 
      En esta actividad se hace una modelación formal de la solución, se busca dividir en sub problemas, donde cada sub problema debe de corresponder a procedimientos o funciones. El objetivo es simplificar el problema, se analizan varias alternativas y se escoge la más viable. Se define el algoritmo o diagrama de flujo que será utilizado para ser implementado en el programa. El algoritmo general debe ser suficientemente específico como para distinguir claramente donde se usan estructuras cíclicas y de control, donde se usan funciones y que hace cada función.
    .
      • Plantear el proceso genérico para solucionar el problema de su proyecto. 
      • Dividir en sub-problemas su estrategia general, es decir simplificar el problema grande dividiéndolo en problemas más pequeños. 
      • Crear el algoritmo correspondiente para cada sub problema.
      • Crear el algoritmo general que mande a llamar a todos los demás algoritmos.
