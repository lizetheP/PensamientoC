Actividad: Ejercicios sobre archivos

Descripción

Desarrolla, en conjunto con el profesor, algunos problemas que implican el uso de archivos de texto.

Objetivo

Aplicar el concepto de archivos de texto en la solución de problemas.

Modalidad y forma de trabajo

Sesión plenaria.

  • Resuelve cualquier duda que te haya quedado del autoestudio previo.
  • Desarrolla e implementa, en conjunto con el profesor, los siguiente problemas.

Problemas

Desarrolla el programa en Python de las siguientes funciones:

  1. La función muestraContenido que recibe la ruta (nombre del archivo de texto) y muestra el contenido en pantalla.

                    muestraContenido(ruta) 

  1. La función copiaArchivo que recibe la rutaOrigen (nombre del archivo de texto origen) y la rutaDestino (nombre del archivo de texto destino). La función debe copiar todo el contenido del archivo origen al archivo destino.

          muestraContenido(rutaOrigen, rutaDestino)

  1. La función cuentaVocales que recibe la ruta (nombre del archivo de texto) y cuente el número de vocales en el archivo de texto y regrese ese valor para desplegarlo.

          cuentaVocales(ruta)

  1. La función menu que imprima el siguiente menú en pantalla:

1. Muestra contenido
2. Copia archivos
3. Cuenta vocales

  1. Desarrolla un programa en Python que despliegue el menú y de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas. Debes de utilizar el ciclo while para que se cicle el programa hasta que el usuario introduzca una opción inválida.

            while True:
                menu()
                opcion = int(input("Introduce una opcion: "))
                if opcion == 1:
                    ruta = str(input(("Introduce el nombre del archivo de texto : "))  # Por ejemplo: prueba.txt
                    muestraContenido(ruta)
                elif opcion == 2:
                    rutaO = str(input(("Introduce el nombre del archivo de texto origen : ")) # Por ejemplo: prueba1.txt
                                        rutaD = str(input(("Introduce el nombre del archivo de texto origen : ")) # Por ejemplo: prueba2.txt
                    copiaArchivo(rutaO, rutaD)
                elif opcion == 3:
                    ruta = str(input(("Introduce el nombre del archivo de texto : "))  # Por ejemplo: prueba.txt
                    res = cuentaVocales(ruta)
                                       print(res)                
                else:
                    print("Opción inválida")
                    break

Guarda tu archivo como: a_matricula.py.
