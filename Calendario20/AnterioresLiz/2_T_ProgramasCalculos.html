Tarea individual: Construcción de programas que requieren cálculos

Ejercicios para desarrollar algoritmos y programas en Python

Desarrollar el algoritmo y programa de los siguientes problemas. 

Modalidad

Individual

Objetivo

Escribir programas simples en Python que impliquen el uso de operadores básicos.

Instrucciones

Desarrolla e implementa, los siguiente algoritmos y programas.

Problemas

  1. Escribe el algoritmo y el programa completo en Python, que solicite 3 valores enteros al usuario, y que realice las operaciones aritméticas mostradas en la parte inferior. El resultado de cada operación debe ser desplegado en la pantalla (procura guardar el resultado de cada operación en un variable auxiliar aux antes de desplegar el resultado de cada operación). Guarda tu archivo como: A1_matricula.py

    Variables utilizadas : a, b, c y aux.

    Operaciones :
    - a / b
    - a % b
    - a / b - c
    - a / (b - c )

    Analiza los resultados de cada operación para los casos de prueba.

    Casos de prueba F(a,b,c):

    - Input: son los valores introducidos por el usuario. Si se ingresan valores de 10, 5, 2; a = 10, b = 5 y c = 2.
    - Output: es el resultado de aplicar las cuatro operaciones respectivamente a los valores de 10, 5 y 2, el resultado esperado para cada caso sería:  2.000,   0,  0.000 y
       3.333

    Input:
    10
    5
    2
    Output:
    2.000
    0
    0.000
    3.333

  2. Escribe el algoritmo y el programa completo en Python que calcule el volumen (v) de un cono recto. Los valores dados por el usuario son el radio (r) y la altura (h). El resultado obtenido (volumen) debe ser desplegado en la pantalla. El valor de PI debe ser declarado como una constante 3.141592, es decir no pidas el valor de PI al usuario. Asegúrate de usar exactamente este valor de PI, ya que si cambias el valor los resultados del cálculo serán diferentes, no pasarán las pruebas. Guarda tu archivo como: A2_matricula.py

Casos de prueba:

- Input: son los valores introducidos por el usuario separados por un espacio . Si se ingresan valores para r = 10 y h = 2.
- Output: es el resultado de la operación. para los inputs 10 y 2, el resultado esperado sería: 209.439

Input:
10
2

Output:
209.439

Especificaciones de entrega

                         Entrega tus archivos en Canvas:

  • Formato de entrega: py
  • Nombre de los entregables: A1_matricula.py y A2_matricula.py
  • Medio de entrega: Se entrega en Canvas en la sección de Actividad 2: Programas con cálculos.
