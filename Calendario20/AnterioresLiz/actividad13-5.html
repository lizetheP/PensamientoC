Actividad: Autoestudio sobre archivos de texto: Archivos.

info_outline Autoestudio sobre archivos de texto: Archivos.

Realizarás el autoestudio que presenta el uso de archivo (de texto y binario) como un medio para almacenar información.

group Modalidad

Individual

check Objetivos de aprendizaje
  • Comprender los conceptos de flujo de datos para crear, leer, escribir y modificar archivos de texto y binarios.
list Instrucciones

  • Estudia de uno de los siguientes libros la secciones indicadas:

    Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2010.

    1. Introducción. (pp. 409)
    2. Flujos. (pp. 410)
    3. Apuntador FILE. (pp. 410)
    4. Apertura de un archivo. (pp. 411)
    5. Funciones de entrada/salida para archivos. (pp. 415)

    Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2014.

    1. Introducción. (pp. 337)
    2. Flujos. (pp. 337)
    3. Apuntador FILE. (pp. 338)
    4. Apertura de un archivo. (pp. 339)
    5. Funciones de entrada/salida para archivos. (pp. 342)
  • Responde las siguientes preguntas:

    1. 1. ¿Qué tipo de archivos se pueden abrir usando C?
    2. 2. ¿Cuál es la diferencia entre archivos .txt .log .c .pl .doc. jpeg .pdf? ¿todos los puedes abrir, todos los puedes crear con C?
    3. 3. ¿Con lo qué sabemos hasta ahora con qué tipo de archivos podemos trabajar en C?
