Actividad: Desarrollando programas con cadenas de caracteres

Descripción

 Desarrollarán la solución de algunos problemas.

Objetivo

Escribir programas en Python que hagan uso de cadenas de caracteres.

Modalidad y forma de trabajo

Diseño colaborativo, implementación individual.

  • Intégrate con tu equipo colaborativo y junto con tus compañeros, analiza cada uno de los ejercicios que se presentan a continuación.
  • Identifiquen, en equipo, cual sería el algoritmo para la solución de cada uno de los ejercicios.
  • De manera individual escribe un programa en Python para cada uno de los ejercicios. Recuerda basarte en el algoritmo que se generó en el equipo.
  • Una vez que hayas terminado tu programa, apoya a tus compañeros de equipo con dudas que tengan (no hagas el laboratorio por ellos ya que esto sólo los perjudica)

Problemas

En un mismo programa desarrolla las siguiente funciones y el script principal:

1. limpiaCadena(caracteresSeguros, cadena)

La función limpiaCadena que recibe 2 cadenas, una de caracteresSeguros y otra que es la cadena a limpiar. La función regresa una nueva cadena que descarta todos los caracteres que no estan en la lista de caracteresSeguros.

Revisa tus notas de python. Usa esta lista de caracteresSeguros, para no tener que pedirla cada vez que corras tu código. Solo pégala en la parte inicial de tu programa y recíbela como parámetro de tu función.

caracteresSeguros = " abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZñÑáéíóú0123456789"       

Casos de prueba:

Input: Número de opción (1. limpiaCadena )
Input: Cadena de texto a limpiar.
Output: La cadena limpia.

Input: 1
Input: Hola! /* Mundo%@
Output: Hola Mundo

Input: 1
Input: Hola! /* Mundo%@
Output: Hola Mundo

Input: 1
Input: Hola! /* Mundo%@
Output: Hola Mundo

2. cuentaOcurrencias(cadena, caracter)

La función cuentaOcurrencias que recibe dos cadenas, la primera contiene una palabra y la segunda contiene el caracter a buscar. La función regresa el numero de veces que el caracter es encontrado en la cadena. 

Casos de prueba:

Input: Número de opción (2. cuentaOcurrencias(cadena, caracter))
Input: Cadena de texto para buscar caracter.
Input: Caracter a buscar en la cadena.
Output: Número de veces que se encuentra el caracter en la cadena.

Input: 2
Input: Hola! /* Mundo%@
Input: o
Output: 2

Input: 2
Input: HOLA Mundo
Input: o
Output: 2

Input: 2
Input: HOLA MUNDO
Input: o
Output: 2

3. obtenNombreCompleto(nombres, paterno, materno)

La función obtenNombreCompleto que regresa el nombre completo de una persona en una cadena de caracteres (separados por espacios), con base a los parámetros recibidos de nombres, paterno y materno.

         Casos de prueba:

Input: Número de opción (3. obtenNombreCompleto(nombre, materno, paterno))
Input: Cadena que contiene los nombres de la persona.
Input: Cadena de caracteres que contiene el apellido paterno de la persona.
Input: Cadena de caracteres que contiene el apellido materno de la persona.
Output: Cadena que contiene el nombre completo de la persona separada por espacios.

Input: 3
Input: Katia Alicia
Input: Godoy
Input: Mendez
Output: Katia Alicia Godoy Mendez

Input: 3
Input: Katia
Input: Godoy
Input: Mendez
Output: Katia Godoy Mendez

4. deletreaCadena(cadena)

La función deletreaCadena que recibe una cadena e imprime cada letra en la cadena separada por un guión medio con excepcioón de la ultima letra, la cual la imprime sin agregar el guión después de ella. Emplea el modificador end='' en el print para evitar que cada print salte de línea, a excepción del último caracter.

          Casos de prueba: 

Input: Número de opción (4. deletreaCadena(cadena))
Input: Cadena a deletrear
Output: Cadena con separación de un guión medio entre cada letra, excepto la última.

Input: 4
Input: Buenos dias
Output: B-u-e-n-o-s- -d-i-a-s

Input: 4
Input: Naofumi
Output: N-a-o-f-u-m-i

                  5. menu() 

La función menu que imprima el siguiente menú en pantalla:

                         1. Limpia cadena
                          2. Cuenta ocurrencias
                         3. Obten nombre completo
                         4. Deletrea cadena

                  6. Script principal

Desarrolla un programa en Python que despliegue el menú y de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas. Debes de utilizar el ciclo while para que se cicle el programa hasta que el usuario introduzca una opción inválida.

caracteresSeguros = " abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZñÑáéíóú0123456789"
 while true:
    menu()
    opcion = int(input("Introduce una opcion: "))
    if opcion == 1:
        c = str(input("Introduce una cadena: "))
        cadena = limpiaCadena(caracteresSeguros, c)
        print(cadena)
     elif opcion == 2:
        c = str(input("Introduce una cadena: "))
        caracter = int(input("Introduce un caracter: "))
        res = cuentaOcurrencias(c, caracter)
        print(res)
    elif opcion == 3:
        nombres = str(input("Introduce el nombre o nombres: "))
        aPaterno = str(input("Introduce el apellido paterno: "))
        aMaterno = str(input("Introduce el apellido materno: "))
        cadena = obtenNombreCompleto(nombres, aPaterno, aMaterno)
        print(cadena)
    elif opcion == 4:
        c = str(input("Introduce una cadena: "))
        deletreaCadena(c)
    else:
         print("Opción inválida")
        break
