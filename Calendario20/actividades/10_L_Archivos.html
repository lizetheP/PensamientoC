TC1028. Pensamiento computacional para ingeniería 
.  

Laboratorio: Archivos de texto
.
.

   Descripción:  
    • Desarrolla, en conjunto con el profesor, algunos problemas que implican el uso de archivos de texto.

 Objetivos:  
    • Aplicar el concepto de archivos de texto en la solución de problemas.
    • Aplicar los conocimientos sobre funciones.

   Modalidad:  
    • Sesión plenaria.
 
 Instrucciones:
  

    • Implementa las siguientes funciones haciendo uso de archivos de texto:

 
   Ejercicios:  

    • La función muestraContenido(nombre) que recibe el nombre del archivo de texto y muestra el contenido en pantalla.
    • La función escribeTexto(nombre) que recibe el nombre del archivo de texto. La función deberá pedir una frase y escribirla en el archivo de texto.
    • La función copiaArchivo(nombreOrigen, nombreDestino) que recibe el nombre del archivo de texto origen y el nombre del archivo de texto destino. La función debe copiar todo el contenido del archivo origen al archivo destino.
    • La función cuentaVocales(nombre) que recibe el nombre del archivo de texto. La función deberá regresar el número de vocales contenidas en el archivo de texto.
    • La función menu() que imprima el siguiente menú en pantalla:

1. Muestra contenido
2. Escribe texto
3. Copia archivo
4. Cuenta vocales
5. Salir

    • La función main() que utilice la función menu para desplegar el menú correspondiente y de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas. Utiliza el estatuto de control if. Recuerda que la captura de datos debe ser realizada en la sección del main. Debes utilizar el ciclo while para que se cicle el programa hasta que el usuario introduzca la opción de salir o una opción inválida.
    • En el script principal manda llamar a la función main.
    • Guarda tu archivo como: K_Matricula.py

                      

 
   Ejercicios:
    • La función escribirEnArchivo(nombre) que recibe el nombre del archivo de texto. La función pide cinco oraciones o frases de menos de 50 caracteres cada frase y las guarde en un archivo de texto.   
    • La función cuentaConsonantes(nombre) que recibe el nombre del archivo de texto. La función deberá regresar el número de consonantes contenidas en un archivo de texto.
    • La función uneArchivos(nombreOrigen, nombreDestino) que recibe el nombre del archivo de texto origen y el nombre del archivo de texto destino. La función deberá agregar al archivo destino el contenido del archivo origen.
    • La función leerDatosArchivo(nombre) que recibe el nombre del archivo de texto. La función deberá imprimir en pantalla los datos del archivo. El archivo contiene los siguientes datos: Nombre ApellidoPaterno Carrera Promedio

Archivo de texto:

    • La función menu() que imprima el siguiente menú en pantalla:

1. Escribir en archivo
2. Cuenta consonantes
3. Une archivos
4. Leer datos archivo
5. Salir

    • La función main() que utilice la función menu para desplegar el menú correspondiente y de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas. Utiliza el estatuto de control if. Recuerda que la captura de datos debe ser realizada en la sección del main. Debes utilizar el ciclo while para que se cicle el programa hasta que el usuario introduzca la opción de salir o una opción inválida.
    • En el script principal manda llamar a la función main.
    • Guarda tu archivo como: K_Matricula.py
