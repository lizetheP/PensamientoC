info_outline Actividad: Ejercicio de Repaso: Archivos.

Desarrollarás la solución de un problema que requiere de la utilización de todos los conceptos vistos en este parcial.

group Modalidad

Individual

check Objetivos de aprendizaje
  • Aplicar los conceptos vistos durante las sesiones pasadas.
list Instrucciones

  • Analiza cada uno de los ejercicios que se presentan a continuación. Identifica cuál sería el algoritmo para la solución de cada uno de los ejercicios. Recuerda documentarlos en tus hojas membretadas.

  • Escribe un programa en Python para cada uno de los ejercicios. Recuerda basarte en el algoritmo que se generaste

  • Entrega, en la sección de "Envío de Tareas", los archivos que contengan los programas en C.

  • Problemas:

    Desarrolla un programa en Python que nos permita jugar ahorcado. El programa deberá trabajar con un archivo de texto llamado "palabras.txt" que contendrá las palabras que se estarán usando para el juego.

    El programa, al inicio de cada juego, debe elegir una palabra al azar del archivo de texto y empezar el juego.

    Adivina la palabra: XXXXXX





    Dime una letra: _

    Cada X representa una letra. Se le preguntará al usuario por una letra. Si el usuario adivina correctamente una letra de la palabra (por ejemplo, el usuario introduce una a) la aplicación deberá mostrar:

    Adivina la palabra: XAXXXA





    Dime una letra: _

    Si el usuario no adivina correctamente (por ejemplo, el usuario introduce x), debe mostrarse la parte del cuerpo apropiada. Después de cada intento de adivinar, debe mostrar todos los intentos del usuario.

    Adivina la palabra: XAXXXA

    O





    Intento 1: X

    Dime una letra: _

    Después de siete intentos fallidos, en la pantalla se desplegará un ahorcado:

    Adivina la palabra: XAXXXA

    O

    / | \

    |

    / \

    Intento 7: XWQZMBP

    La palabra es: CADENA

    Si el usuario adivina correctamente, la aplicación deberá mostrar:

    Adivina la palabra: CADENA

    O





    Intento 1: X

    ¡¡¡Felicidades!!! Has podido adivinar la palabra. Deseas jugar otra vez (si/no)?

    Debe introducirse la respuesta "si" o "no" apropiada.

    Incluye el algoritmo como comentario al inicio del programa.
