Actividad: Desarrollando programas con cadenas de caracteres

Descripción

 Desarrollarán la solución de algunos problemas.

Objetivo

Escribir programas en Python que hagan uso de cadenas de caracteres.

Modalidad y forma de trabajo

Diseño colaborativo, implementación individual.

  • Intégrate con tu equipo colaborativo y junto con tus compañeros, analiza cada uno de los ejercicios que se presentan a continuación.
  • Identifiquen, en equipo, cual sería el algoritmo para la solución de cada uno de los ejercicios.
  • De manera individual escribe un programa en Python para cada uno de los ejercicios. Recuerda basarte en el algoritmo que se generó en el equipo.
  • Una vez que hayas terminado tu programa, apoya a tus compañeros de equipo con dudas que tengan (no hagas el laboratorio por ellos ya que esto sólo los perjudica)

6. Script principal

Desarrolla un programa en Python que de acuerdo a la opción seleccionada por el usuario le dé la oportunidad de ejecutar cualquiera de las funciones que han sido construidas.

caracteresSeguros = " abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZñÑáéíóú0123456789"
opcion = int(input())
if opcion == 1:
    c = str(input())
    cadena = limpiaCadena(caracteresSeguros, c)
    print(cadena)
elif opcion == 2:
     c = str(input())
    caracter = int(input())
    res = cuentaOcurrencias(c, caracter)
    print(res)
elif opcion == 3:
    nombres = str(input())
    aPaterno = str(input())
    aMaterno = str(input())
    cadena = obtenNombreCompleto(nombres, aPaterno, aMaterno)
    print(cadena)
elif opcion == 4:
    c = str(input())
    cadena = deletreaCadena(c)
    print(cadena)
else:
     print("Opción inválida")

  • No olvides verificar que tu programa imprima el número de decimales indicados en los Casos de prueba. (Modifica el código que se te muestra para que imprima los decimales correspondientes)
