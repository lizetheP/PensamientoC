Actividad: Ejercicios sobre el ciclo while

Descripción

Desarrolla, en conjunto con el profesor, algunos problemas que implican el uso de un ciclo while.

Objetivo

Aplicar el ciclo while en la solución de problemas.

Modalidad y forma de trabajo

Sesión plenaria.

  • Resuelve cualquier duda que te haya quedado del autoestudio previo.
  • Desarrolla e implementa, en conjunto con el profesor, los siguiente problemas.

Problemas

Desarrolla el algoritmo y el programa en Python de las siguientes funciones:

  1. La función división que recibe dos números enteros positivos num1 y num2 y calcula la división entera y el residuo de num1 entre num2 utilizando el ciclo while y la operación resta.

Por ejemplo:

  1. La función inverso que recibe un número entero y devuelve su inverso numérico.

Por ejemplo:

    • inverso(1234) = 4321

    • inverso(45678) = 87654

  1. Calcular el N-ésimo número de Fibonacci. Usa la relación de recurrencia FN = FN-1 + FN-2, con los dos primeros números de la secuencia F1 y F2 definidos como 1. Usando esta relación de recurrencia, se pueden calcular los primeros 7 números de Fibonacci como sigue:

Fib(1) = 1
Fib(2) = 1
 Fib(3) = Fib(1) + Fib(2) = 1 + 1 = 2
Fib(4) = Fib(2) + Fib(3) = 1 + 2 = 3
Fib(5) = Fib(3) + Fib(4) = 2 + 3 = 5
Fib(6) = Fib(4) + Fib(5) = 3 + 5 = 8
Fib(7) = Fib(5) + Fib(6) = 5 + 8 = 13
