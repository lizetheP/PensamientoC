Funciones

Objetivo

Aplicar los conceptos de función.

Revisión de autoestudio

1. ¿Qué aparece en la pantalla si se ejecuta el siguiente código de Python?

def uno (a, b) :   
              a = a + 5     
              b = b + 10     
              print(a)     
              print(b)     
              return b

x = 5
y = 8
z = uno(x, y)
print(x)
print(y)
print(z)

2. ¿Qué aparece en la pantalla si se ejecuta el siguiente código de Python?

# nota que el print de esta función muestra el valor
# de a, un espacio en blanco y luego el valor de b

def func1 (a, b):      
     print(str(a) + " " + str(b))

def func2 (x, y):      
     x = 10      
     y = y * 2       
     func1 (y, x)      

p = 2
q = 3
func2(p,q)
func1(p,q)
